# 🎨 Ding Dong Doodle

A creative platform where kids bring their hand-drawn characters to life, build worlds, and star in their own stories.

---

## What It Does

- **Draw or Upload** — Sketch directly in the browser or upload photos of paper drawings
- **Create Islands** — Build custom island worlds where characters live
- **Animate Characters** — Characters walk, jump, wave, and explore their islands
- **Give Personalities** — Assign traits (brave, silly, tidy, adventurous) that shape stories
- **Draw Evolutions** — Create new versions of characters over time, building a visual timeline
- **Generate Stories** — Personalized, illustrated stories starring the child's own characters

---

## Tech Stack

- **Frontend**: React, HTML5 Canvas
- **Animation**: Custom sprite-based animation system
- **Story Generation**: Claude API (Anthropic)
- **Island Builder**: Tile-based world editor in HTML5 Canvas
- **Backend**: Node.js + Express
- **Storage**: Cloud storage for drawings, character data, and story history

## Getting Started

### Prerequisites

- Node.js
- npm or yarn
- Anthropic API key

### Installation

```bash
git clone https://github.com/your-username/ding-dong-doodle.git
cd ding-dong-doodle
npm install
```

### Run Locally

```bash
npm run dev
```
